# phpstats

phpstats — это небольшая утилита для сбора статистики проектов на PHP.

### Установка

```
go get github.com/i582/phpstats
```

### Использование

```
collect <dir>
```

После сбора информации вы попадете в интерактивную оболочку, для помощи введите `help`.



### Roadmap
